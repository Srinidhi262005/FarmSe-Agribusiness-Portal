# FarmSe Agribusiness Portal
FarmSe is a full-stack web-based agribusiness platform designed to connect farmers and buyers through a digital marketplace. It eliminates middlemen, enabling farmers to sell crops directly while buyers can easily browse and purchase agricultural products.
The platform also integrates AI-based crop prediction and real-time weather insights to support smarter farming decisions and improve productivity.

##  Features
- 👤 User Authentication (Farmer, Buyer, Admin)
- 🌾 Crop Listing & Management
- 🛒 Marketplace for buying and selling crops
- 🔍 Search and filter crops
- 💬 Real-time communication between users
- 🤖 AI-based Crop Prediction
- 🌦 Weather API integration
- 🛠 Admin dashboard for system control
- 📱 Responsive UI design


##  Tech Stack
- Backend:Flask (Python)
- Frontend:HTML, CSS, Bootstrap
- Database:SQLite 
- Others:** Jinja2, REST APIs, Machine Learning

## Objectives
-Enable direct farmer-to-buyer interaction
-Provide a digital marketplace for agriculture
-Use AI to guide crop selection
-Improve farmer income and reduce dependency on intermediaries
